$whoami

I come from a professional background in restaurant management and information technology, and currently work in IT for a restaurant data company.

My first serious venture into the tech world started with building a PC. I spent months researching and learning all about computer hardware. My research sparked an interest in Linux, and it wasn't long after that I began modifying distributions and attempting to automate daily tasks with shell scripting. Linux led to shell scripting, shell scripting led to learning other programming languages, and I soon developed a passion for programming and information security.


Languages and Frameworks

Bash, Zsh, Powershell, Python, PHP, SQL, HTML, and CSS, and JavaScript.


Tools I use

Visual Studio Code is my main code editor. I am also familiar with Atom, Sublime Text, IDLE, and Visual Studio IDE among other editors. I use GitHub to store my code and work on projects there regularly and am also familiar with using command-line Git.

I mostly use Kali Linux for hacking, but I've used Parrot OS in the past as well. As far as hacking tools go, I'm familiar with Nmap, John the Ripper, Aircrack-ng, Wireshark, Burp Suite, the Social Engineering Toolkit, Metasploit, among others.


Courses I've Been Taking

  • CS50 taught by David Malan from Harvard University edX
    •

    An introductory course on computer science taught at Harvard University and available online to audit for free.

  • Automate the Boring Stuff with Python Programming by Al Sweigart
    •

    Book and online course taught by the author that covers the basics of Python programming, automating tasks, debugging, web scraping, and installing third-party modules among other things.

  • IT Security Foundations: Operating System Security by Lisa Bock
    •

    Online course covering foundational concepts of operating system security and how to apply those security concepts to your own operating systems.

  • MITRE ATT&CK Defender Fundamentals from Cybrary
    •

    Cybrary course covering the use of MITRE ATT&CK and developing a threat-informed mindset to understand how defenses actually perform against real-world attacks.

  • macOS Support: Network Services from Pluralsight
    •

    Pluralsight course covering how to configure network services, connect to file services, manage sharing services, configure firewalls, and troubleshoot network problems on macOS.

  • Learn Python and Ethical Hacking from Scratch by Zaid Sabih
    •

    Course taught by Zaid Sabih from zSecurity that covers setting up a virtual lab for penetration testing as well as writing programs and scripts in Python such as MAC address spoofers, network scanners, ARP spoofers, packet sniffers, DNS spoofers, file interceptors, and code injectors.


Recent Reading

  • Information Technology Essentials by Eric Frick
    •

    Covers subjects such as computer hardware, how software is built, computer security, and cloud computing.

  • CompTIA A+ Certification Exam Guide by Mike Meyers
    •

    An in-depth guide to studying for the CompTIA A+ certification and related subjects.

  • The Art of Deception by Kevin Mitnick
    •

    An in-depth look at social engineering that examines the human factors of information security and how organizations can protect themselves from these attacks.

  • You'll See This Message When It Is Too Late by Josephine Wolff
    •

    An analysis of a series of cybersecurity incidents. Wolff discusses the entire process of each breach in order to identify what could have been done differently, and where defenses could have been better. She outlines motives for cyber attacks and teaches the lessons that can be learned from them to create stronger and more effective security in the future.

  • Ghost in the Wires by Kevin Mitnick and William L. Simon
    •

    Memoir of hacker and cybersecurity expert Kevin Mitnick.

  • Permanent Record by Edward Snowden
    •

    Autobiography of former CIA and NSA contractor Edward Snowden.

Outside of Tech
