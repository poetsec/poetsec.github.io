Projects

Here are a few of the projects I've been working on. I also contribute to open source wherever I can and write about hacking, security, and Linux on my blog.


MAC Address Changer

This command-line MAC address spoofer allows the user to mask their MAC address from the command-line. It asks the user to specify an interface, choose a new MAC address, and then masks their real MAC addresss with the new, specified one. This was written as part of a course on ethical hacking and Python concepts by Zaid Sabih. See code.


Network Scanners

I created a set of network scanners in Python using the Scapy module. One is a simple scanner that uses scapy.arping to perform the network scan, and the other is a more customized scanner that has a manual scan function written out step by step, uses dictionaries to organize the information it collects, and displays that information to the user in an easy to read table. These were both created as part of a course on ethical hacking and Python concepts by Zaid Sabih. See code.


#LinuxHelp Twitter Bot

I created a Twitter bot in Python using the tweepy module. The bot retweets any tweets that contain the hashtags #LinuxHelp or #LinuxQuestion. The idea of the bot is that Twitter users with Linux related questions can use the hashtags for their tweet to get more visibility, and Twitter users who are interested in answering Linux related questions can follow the bot to have questions retweeted to their feed. Learn more.


File Encypter/Decrypter

This is a bash shell script for encrypting and decrypting files. When run, it gives the user the option to either encrypt or decrypt a file that they select using GPG. See code.


Random Password Generator

This is a bash shell script for generating random passwords. It prompts the user for the number of characters they would like their new password to be, and then generates five random passwords. See code.


Cash Drawer Calculator

This is a work-in-progress program to help cashiers balance tills. It allows the user to enter the number of each cash denomination they have, and then does the math to come up with the total dollar amount. It's based on a Microsoft Excel sheet I've used in the restaurant industry, and I plan on making improvements in the future such as adding options for foreign currencies and a GUI. See code.
